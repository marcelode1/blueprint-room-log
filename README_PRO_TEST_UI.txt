ProjectONus Pro Test UI

The classic ProjectONus interface is still the default.

To test the new desktop style:
1. Log in as admin from a desktop browser.
2. Click "Pro Test UI" in the left/top navigation, or open /ui/pro-test.

To go back:
1. Click "Classic UI", or open /ui/classic.

The test styling is kept separately in:
static/pro_interface_test.css

This theme is inspired by modern project-management software patterns, not a
copy of any company's branding, images, or exact layout.
